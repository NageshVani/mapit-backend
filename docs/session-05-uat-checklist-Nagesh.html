This checklist covers the two things built in Session 5: (1) sellers now get an email the moment a buyer taps "I'm Interested" and writes a note, and (2) that note now has a small typing box instead of sending a fixed canned message. The automated checks (21 code/API/DB checks) have already passed, including one real bug that was found and fixed today — Block C below specifically re-tests that fix.

You'll need two accounts to test this properly: one to act as the buyer (view someone else's listing and tap "I'm Interested"), and access to the seller's email inbox to check what arrives. Block B — The New "I'm Interested" Note Box

B1
Tapping "I'm Interested" opens a small typing box, not an instant send
  1. Sign in and browse to any listing that isn't your own
  2. Tap the orange 🤝 I'm Interested button
  3. Verify: a small text box appears below, with faint grey placeholder text like "Hi! I'm interested in your listing…" — nothing sends yet
B2
"Send" button stays greyed out until you actually type something
  1. With the note box open, look at the Send button before typing anything
  2. Verify: Send is greyed out / not clickable while the box is empty
  3. Start typing a few words
  4. Verify: Send turns solid green and becomes clickable as soon as you type
  5. Delete everything you typed
  6. Verify: Send greys out again when the box is empty
B3
Cancel closes the note box without sending anything
  1. Open the note box, type something, then tap Cancel
  2. Verify: the box closes, the "I'm Interested" button is still there and untouched (no "Sent!" shown)
B4
Sending a real note works and shows confirmation
  1. Open the note box on a listing, type a short real message (e.g. "Is this still available? Can I see it this weekend?")
  2. Tap Send
  3. Verify: a green toast appears saying the message was sent, the "I'm Interested" button changes to "✅ Sent!" in green, and it's no longer clickable

Block C — Re-test: "I'm Interested" from the Saved tab (today's bug fix)

C1
Saving a distant listing, then messaging it from the Saved (❤) tab, works correctly
  1. Increase your search radius, find a listing that's far from your usual home area, and tap the heart (♡) to save it
  2. Now set your radius back down small, or switch category, so that listing would no longer show up in your normal Browse results
  3. Go to the ❤ Saved tab and open that same listing
  4. Tap 🤝 I'm Interested, type a short note, and tap Send
  5. Verify: it sends successfully with the green "✅ Sent!" confirmation — no "Listing not found" error. (This exact scenario was broken before today's fix.)

Block D — Seller Notification Email

D1
Seller receives an email with the buyer's actual typed note
  1. As the buyer, send an "I'm Interested" note (as in B4) on a listing owned by an account whose inbox you can check
  2. Check that seller's email inbox within a minute or two
  3. Verify: an email arrives with subject mentioning the buyer's name and the listing title, and the body shows the exact words you typed in the note box (not the old generic canned message)
D2
Email shows the "unscreened" warning (expected, since no Anthropic key yet)
  1. Open the email from D1
  2. Verify: subject line starts with [Unscreened] and the email body shows a small orange banner saying the message hasn't been automatically verified as genuine buyer interest yet
  3. This is expected/correct for now — it will go away automatically once Arun approves adding the Anthropic key and real spam-checking turns on. Not a bug.
D3
The message also still appears normally in the seller's in-app Inbox
  1. Sign in as the seller in the app itself (not email)
  2. Open the profile menu → Inbox
  3. Verify: the buyer's note shows up as a normal unread message, exactly as before — the email is an extra notification, not a replacement for in-app messaging
D4
Only the buyer's very first message on a listing triggers the email
  1. After D1's email arrives, reply again to the same listing/seller (e.g. via the in-app chat box)
  2. Verify: the seller does not get a second notification email for the follow-up message — only the very first "I'm Interested" note triggers it

Block E — Note Box Limits

E1

  1. Open the note box and try pasting or typing a very long message (500+ characters — e.g. paste a paragraph from a website)
  2. Verify: typing simply stops accepting more characters around 500 — it doesn't let you keep going indefinitely, and doesn't crash or error
