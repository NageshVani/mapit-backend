B3

  1. Open My Ads tab after posting (same listing from B2)
  2. Verify: the listing card shows "Expires in 30d" (or 29d if just past midnight) in grey/muted colour below the status dot
  3. Verify it is NOT orange or red (a brand-new listing should have ~30 days remaining)
To verify the warning colours (optional): open Supabase SQL Editor and run: UPDATE listings SET expires_at = NOW() + INTERVAL '4 days' WHERE title = 'your test listing title'; Reload My Ads — the countdown should turn orange and show "⚠️ Expires in 4d".

Then run: UPDATE listings SET expires_at = NOW() - INTERVAL '1 day' WHERE title = 'your test listing title'; Reload — should show ⚠️ Expired in red. Restore with: UPDATE listings SET expires_at = created_at + INTERVAL '30 days' WHERE title = 'your test listing title';
